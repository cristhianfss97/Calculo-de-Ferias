# Calculo-de-F-rias
Desenvolvi essa aplicação para realizar cálculos de férias de funcionários, levando em consideração regras da CLT, como período aquisitivo, férias proporcionais, adicional constitucional de 1/3 e descontos legais. Projeto focado em lógica de negócio, organização de código e boas práticas de desenvolvimento, além de servir como base para aprendizado
